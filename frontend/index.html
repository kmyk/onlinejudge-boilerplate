OnlineJudge Boilerplate / AtCoder の問題の URL を入力するといい感じのテンプレを出力してくれる Web ページ

source
説明
  • 入力を受け取る部分や MOD などを自動で生成し、あとは solve() 関数を埋めるだけないい感じのテンプレが生成されます。
  • 変数 test_samples を true にするとサンプルケースを自動でテストしてくれます。ただし、答えが合っているかどうかは目diffし、提出前には false に戻すことを忘れないでください。
  • 内部では atcoder-tools と online-judge-tools をいい感じに叩いています。
  • C++ 以外の言語についてのテンプレが欲しいなら atcoder-tools を、もっと楽にサンプルケースをテストしたいなら online-judge-tools を直接使ってください。
